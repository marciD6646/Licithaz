## 📖 Projekt leírása

Az **Licitház** egy Laravel alapú webalkalmazás, amely lehetővé teszi licitre bocsátott termékek böngészését és megvásárlását.

A cél egy modern, könnyen kezelhető aukciós rendszer létrehozása, amely leváltja a jelenleg sok esetben elavult és nehezen kezelhető platformokat.
